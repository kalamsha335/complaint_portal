# ✨ Overview

The **Complaint Portal** is a full-stack web application built to streamline the complaint management process for organizations, institutions, and businesses.

Users can easily submit complaints, monitor their status in real time, and receive updates, while administrators can efficiently manage, assign, and resolve complaints through an intuitive dashboard.

# ⚡ Getting Started

## Clone Repository

```bash
git clone https://github.com/kalamsha335/Complaint-Portal.git
```

## Navigate to Project

```bash
cd Complaint-Portal
```

## Move to XAMPP

Copy the project into

```
xampp/htdocs/
```

---

# 🗄 Database Setup

Create a MySQL database named

```
complaint_portal
```

Import the provided SQL file.

Configure your database connection inside

```php
config/database.php
```

```php
$host="localhost";
$user="root";
$password="";
$dbname="complaint_portal";
```

---

# ▶ Run the Application

Start

- Apache
- MySQL

using **XAMPP Control Panel**

Open

```
http://localhost/Complaint-Portal
```
